# 二手交易管理系统简介 java317

## 项目概述

二手交易管理系统是一个基于Java和SpringBoot框架开发的前后端分离的Web应用程序。该系统旨在为用户提供一个便捷、高效的二手物品交易环境，实现用户之间安全可靠的交易。

## 技术栈

- **开发语言：** Java
- **前端框架：** Vue
- **后端框架：** SpringBoot
- **数据库：** Mysql
- **持久层框架：** Mybatis
- **构建工具：** Maven

## 系统角色

- **普通用户：** 可以发布物品、查看消息、管理个人中心等。
- **管理员：** 负责系统管理，包括物品管理、订单管理、用户管理等。

## 运行环境

- **操作系统：** Windows / Mac
- **开发工具：** IDEA
- **JDK版本：** 1.8及以上

## 功能模块

### 前台功能

1. **首页**
   - 全部查询
   - 分类查询
   - 关键字搜索
2. 发布物品
3. 查看消息
4. **个人中心**
   - 编辑个人信息
   - 编辑收货地址
5. 状态查询
   - 我发布的
   - 我下架的
   - 我收藏的
   - 我卖出的
   - 我买到的

### 后台功能

1. **交易物品管理**
   - 上线的二手物品（可以违规下架）
   - 下架的二手物品（可以永久删除）
2. 订单信息管理（删除）
3. **用户信息管理**
   - 正常用户（可封号）
   - 违规用户（可以解封）
   - 管理员（可添加管理员）

## 核心亮点

- **灵活的系统架构：** 采用B/S架构和MVC模式，确保系统可扩展性和可维护性。
- **便捷的开发环境：** 使用IDEA、JDK1.8等现代开发工具和版本，提高开发效率。
- **全面的交易管理：** 提供从前台发布到后台管理的全流程交易功能，覆盖用户交易的各个场景。
- **严格的安全控制：** 实现对用户和物品的多维度管理，保障交易安全。
